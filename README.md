# vxe-table
基于vue3+ts+vxeTable二次封装的表格组件

vxe-table地址：https://vxetable.cn/v4/#/component/table/custom/fixed

项目初始化后安装vxe-table相关依赖：https://vxetable.cn/v4/#/start/useUI/install
我们是全量引入的，省心

基于当时的项目需求，过滤部分可忽略，进上传table和过滤部分，可直接在自己的项目中使用，具体视项目需求

    margin: 20px !important;
    margin-bottom: 0 !important;
    height: calc(100vh - 117px) !important;
}
